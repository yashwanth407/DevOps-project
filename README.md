# 💎 Glass Tax Calculator

A modern and elegant **Tax Calculator** built using HTML, CSS, and JavaScript with a stunning **Glassmorphism UI** design.

---

## 🚀 Features
- 💰 Calculates tax amount and total instantly.
- ⚡ Real-time updates as you type.
- 🧊 Glassmorphism design with smooth hover effects.
- 📱 Fully responsive layout for mobile and desktop.
- ♿ Accessible (keyboard and ARIA friendly).
